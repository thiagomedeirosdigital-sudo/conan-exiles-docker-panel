# Contribuindo

Obrigado por contribuir com o Conan Exiles Docker Panel.

Você pode ajudar com:

- correções de bugs
- melhorias de documentação
- novos recursos
- testes em outros ambientes
- melhorias de segurança

Antes de abrir Pull Request, confirme que não incluiu senhas, .env real, bancos, backups, logs privados ou mods baixados.
